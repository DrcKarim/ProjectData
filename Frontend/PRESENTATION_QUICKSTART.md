## Step 3: Add More Slides (2 minutes)

### Key Insight Slide

Perfect for big numbers:

1. Add new slide, select type **"key-insight"**
2. **Title**: "Record Revenue"
3. **Metric Value**: "$2.5M"
4. **Metric Label**: "Q4 Revenue"
5. **Insight Text**: "Exceeded target by 30% driven by enterprise sales"

### Text Slide

For bullet points:

1. Add new slide, select type **"text"**
2. **Title**: "Key Achievements"
3. Click **"+ Add Bullet"** to add points:
   - "Revenue up 45% YoY"
   - "Customer retention at 92%"

### Conclusion Slide

Wrap it up:

1. Add new slide, select type **"conclusion"**
2. **Title**: "Next Steps"
3. **Summary**: "Strong Q4 positions us for growth in 2026"

## Common Workflows

### Quick Demo (1 slide)

```javascript
1. Create 1 title slide
2. Fill in title and subtitle
3. Click "Present"
4. Fullscreen automatically
5. Press Esc when done
```

### Standard Presentation (5-10 slides)

```javascript
1. Title slide
2. Agenda/overview (text slide)
3. Main content (2-3 chart slides)
4. Key insight slide
5. Recommendations (text slide)
6. Conclusion slide
```

```javascript

8. Key insights
9. Conclusion
```

## Tips for Success

### Design Tips

âœ… **DO:**
- One key message per slide
- Large fonts (readable from back of room)
- High contrast colors
- Minimal bullet points (3-5 max)
- Consistent chart styling

âŒ **DON'T:**
- Cram too much information
- Use tiny fonts
- Over-animate
- Read slides word-for-word
- Ignore speaker notes

### Presentation Tips

**Before:**
1. Practice with keyboard shortcuts
2. Test fullscreen on actual display
3. Write speaker notes for each slide
4. Time yourself (aim for 1-2 min/slide)

**During:**
1. Make eye contact with audience
2. Use annotations to guide attention
3. Pause for questions
4. Watch the timer
5. Keep steady pace

**After:**
1. Exit with Esc key
2. Return to builder to make edits
3. Export presentation for sharing

## Themes Quick Reference

Change presentation theme in builder settings:

**Dark** (Default)
- Best for: Evening, tech audiences
- Background: Dark blue/gray
- Accent: Bright blue

**Light**
- Best for: Morning, printed handouts
- Background: White
- Accent: Blue

**Executive**
- Best for: C-level, board meetings
- Background: Slate gray
- Accent: Green

**Corporate**
- Best for: Formal settings, investors
- Background: Navy
- Accent: Deep blue

## Troubleshooting Quick Fixes

### Fullscreen not working?
**Fix**: Press F key or click Fullscreen button
**Reason**: Must use user action, not automatic

### Slides not saving?
**Fix**: Changes auto-save. Refresh to verify.
**Reason**: Zustand persist stores in localStorage

### Keyboard shortcuts not working?
**Fix**: Click anywhere on slide first
**Reason**: Input fields capture keystrokes

### Annotations not showing?
**Fix**: Press A key to toggle them on
**Reason**: May have been toggled off
